## Features
* Đặt phòng
* Nhận phòng
* Thêm dịch vụ
* In hóa đơn
* Quản phòng và loại phòng
* Quản lý dịch vụ và loại dịch vụ
* Quản lý nhân viên
* Phân quyền
* Báo cáo doanh thu theo từng loại phòng
* Thay đổi quy định

## Deployment
* .NET Framework 4.6.1
* SQL Server 2017
